# CSR Upcycling - Odoo Module

An Odoo app that turns end-of-life inventory into a measurable sustainability pipeline.  
Employees log upcycle requests, managers approve them, and the system tracks CO₂e avoided, AED saved, and CEITs earned —  
making sustainability measurable and rewarding.

**Features**
- Upcycle request workflow with approvals.
- Computed CO₂e, AED, and CEITs metrics.
- Product & department analytics.
- Rewards and leaderboard gamification.
- Pivot, graph, and kanban views for reporting.

**Tech Stack**
- Odoo (Python, XML)
- PostgreSQL
- HR, Product, Stock integration

**Installation**
1. Clone this repo into your Odoo addons path.
2. Update apps list and install "CSR Upcycling".
